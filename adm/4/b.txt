# ADM4 Software Stack

Learn about software stack and how to set them up.
Differentiate between system and user based software stack.
Also, learn about using containers for software stacks.

## Requirements

## Learning Outcomes

* Review how to set up a software stack and remember system package managers, building from source, and a module system
* Understand how to use containers and their deployment, as well as software containerisation
* Recognize how user can manage their own software by using HOME Directory installs, SPACK, and similar environments
