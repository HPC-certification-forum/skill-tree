# K1.3 Input/Output

The hardware/software organization/structure in a data center/supercomputer.

## Learning Outcomes

* Analyse different IO Layers.
* Demonstrate different data reduction techniques.
* Outline data management with cache coherence or staging.
* Outline access patterns for files and also parallel file access.
