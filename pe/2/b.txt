# PE2 Measuring System Performance

## Learning Outcomes

* Measure the system performance by the help of standard tools and by profiling in order to assess the runtime behavior of parallel programs.
* Compute the utilization of a machine by various measures.
* Discover the IO performance of a program and select an appropriate model.
