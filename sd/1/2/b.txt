# SD1.2 Parallel Programming

Parallel programming of shared memory systems and message passing systems as well as load balancing is addressed.

## Learning Outcomes

* Assess the parallel nature of algorithms.
* Develop parallel programs using Shared memory programming.
* Develop parallel programs using Message passing.
* Analyse load balancing algorithms and paradigm.
* Demonstrate OP programming middleware.
* Demonstrate GPU programming using CUDA
* Breakdown the difference of accelerators compared with a CPU
