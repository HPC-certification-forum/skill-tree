# USE1.4.3-B Spack

# Background

Spack is a package manager for Linux and MacOS.
It allows to install software with all necessary dependency and to load relevant software.
Spack creates unique configurations for software.

# Aim
  * To be able to use the spack command to load software.

# Outcomes
  * Describe the concept of configurations and used hashing
  * Querying information about packages
    - list
    - info
    - find (also use -d to reveal their dependencies)
    - spec
  * Loading/Unload software modules
  * Describe the spec syntax to specify package configurations
